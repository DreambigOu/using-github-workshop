Using GitHub for Scholarly Publishing and Collaboration

a Savvy Researcher Workshop

The workshop provides an overview of navigating GitHub and interacting with open source repositories. Included is a hands-on tutorial that demonstrates publishing a basic blog using GitHub Pages as a web host, and Jekyll as a static website generator.
